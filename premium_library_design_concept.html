# Book Website UI/UX Concept: "Premium Library" ## 🎨 1. Color Palette - **Primary (Soft Purple):** #8B5CF6 - **Secondary (Pastel Pink):** #F472B6 - **Accent (Sky Blue):** #38BDF8 - **Background:** #F9FAFB (Light Gray) - **Card Background:** #FFFFFF - **Text Primary:** #1F2933 - **Text Secondary:** #6B7280 - **Brand Identity Gradient:** `linear-gradient(135deg, #8B5CF6, #F472B6)` ## 🧠 2. Design Style - **Keywords:** Soft, Clean, Calm, Friendly, Premium - **Elements:** Soft shadows, rounded corners (xl), glass/smooth cards, minimal + spacious. ## 📚 3. Public Website UI - **Homepage:** Navbar with gradient logo, Hero section with "Discover Your Next Favorite Book 📖", and a Book Grid featuring rounded-2xl cards with hover glow effects. - **Book Detail Page:** Layout with cover, title, author, description, and a gradient download button. ## 🔐 4. Admin Dashboard UI - **Layout:** Sidebar (dark purple gradient) + Main Content. - **Dashboard Page:** Stats cards (Total Books, Recent Uploads, Storage) with soft gradient backgrounds. - **Add Book Page:** Form with rounded inputs, light borders, and purple focus glows. ## 🧃 5. Micro Interactions - Button hover: glow + scale. - Input focus: soft purple ring. - Card hover: lift + shadow. - Smooth transitions (0.2–0.3s). ## 🧩 6. Component Style Guide - **Buttons:** Primary (Gradient), Secondary (Outline Purple), Danger (Soft Red). - **Cards:** Rounded-xl/2xl, Padding p-4 or p-6. - **Inputs:** Border #E5E7EB, Focus purple ring. - **Special Features:** Floating category bar (chips), Smart search UX, Featured books section.
